# CoolEditor
A Java Swing , and Java AWT based text editor  with operations like:
1.	Saving a file 
2.	Deleting a file 
3.	Opening a new file , browsing 
4.	Changing font color 
5.	Changing background color 
6.	Exiting the system 
7.	Opening a new window 
8.	Editing 
9.	Cut, Copy , Paste
